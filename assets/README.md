# Delos Lounge & Dining — Brand Asset Pack

## Recommended usage

### Primary website logo
Use `website/delos-logo-primary.jpeg` for the main navigation and footer.
Best placement: gold logo on a deep charcoal or black background.

### Favicon / small icon direction
Use the lion-head direction from `website/delos-lion-lockup.jpeg`, but crop/simplify it before final production.
The full wordmark is too detailed at favicon size.

### Secondary brand seal
Use `website/delos-secondary-seal.jpeg` as a premium seal/watermark.
Best uses: menu cover, section background, loading screen, Instagram highlights, packaging stickers.

### Hero candidates
Use the files inside `hero-candidates/` when deciding the homepage hero direction.
Recommended hero approach: use real restaurant food/lounge photography as the main hero, then use one Delos asset as a subtle overlay or watermark.

Best hero asset options:
1. `hero-candidate-full-greek-lion-illustration.jpeg` — strong launch/poster visual; too busy for permanent navbar use.
2. `hero-candidate-circular-seal.jpeg` — best subtle luxury watermark.
3. `hero-candidate-lion-head-lockup.jpeg` — good for brand-forward hero sections.
4. `hero-candidate-wordmark-stone-lion.jpeg` — usable for an about/heritage section.

## File structure

- `originals/` — original supplied logo images.
- `website/` — practical website selections.
- `favicon/` — placeholder folder for final favicon exports after icon cleanup.
- `hero-candidates/` — candidates for homepage hero testing.
- `social-print/` — larger assets for posters, social announcements, and menu covers.

## Implementation note
Use the clean wordmark as the main identity. Use the lion head as the recognition symbol. Use the circular Greek seal as a decorative luxury asset.

## Next production step
Before final website implementation, convert the selected primary logo and icon to transparent SVG/PNG files. The supplied images are JPEGs, so they still have white backgrounds.
